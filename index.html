formir

Pure CSS & HTML5 Framework
for
responsive web design developing

Why formir

  • minimize Java Script code in application,
  • no more !important - we give you free hand with style,
  • use less style and html as possible,
  • give you able to set every class your own name,
  • propose you to use structure of files gained of our experience,
  • do it all with the help of modules and mixins.
