formir

Formir is open-source pure CSS Framework
for responsive web developing

Based on flexbox, complete and modular, fluid design,
builded with Less. No-JS, no more override !important.

Why formir

  • not need JS for fully work
  • customizable class name
  • easy to modify
  • clean and smart html code
  • object oriented css
  • clean files and mixing structure
  • mixing and helper class ready to use
  • ready for JS framework and React
  • fully RWD support
  • sidebar included
  • modular structure
