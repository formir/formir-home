Grid pure css responsive grid system based on flexbox with configurable gutter size and columns count

Grid system is most powerfull components in this framework.

To apply grid just add grid class to parent of nodes, nodes will have class cell with number of columns to span, example: cell-3. Maximum number of columns you can set on setup. Also if you leave only cell class without number this column will be auto width to rest of available space. There is also two more property of width of cell cell-fill and cell-auto, see more belov.

Gris system has vertical and horizontal space between cell, we called gutter and space, gutter is space between column, and space is between row. You can dinamicly changes this space using extra class after or before grid.

Cell wil auto wrap, so if you want to be sure that thay will be separata by new row, just wrap them by new parent div with class row.

Space and gutter will always the same, no matter if cell will wrap automaticly or wrap by row.

You could also change the width of cell by responsive class, breaking points are set on variables. You could user prefix t-cell-{number} - tablet, tp-cell-{number} - tablet portrait, m-cell-{number} - mobile, mp-cell-{number} - mobile portrait

One more thing: offset-{number} or offset-right-{number} use this class on cell to set offset (margin) on right and left side to cell, works also with RWD prefix.
