Tabs

Tabs and navigation is control by input tag, id of seach input should be unique.

navigation label should be set te same for as id of his input in the same order.

You could use any tag to list your pages to display, but do not change display priperty of this elements.

Before you start, maximum number of supported tab set on core setup file, (default set to 10)

You can combine any other components with tabs, buttons, section, menu, grid etc.
